# Clinic — Estética & Saúde

Site institucional da **Clinic — Estética & Saúde** (Sombrio/SC).
Odontologia estética, harmonização facial e medicina canábica.

🌐 **Produção:** https://clinicsombrio.com

## Tecnologia
Site estático (HTML, CSS e JavaScript puro) — sem build, sem dependências.
Hospedado na **Vercel**, com deploy automático a cada `git push` para a branch `main`.

## Estrutura
```
index.html          Página principal (vídeo no hero)
canabidiol.html     Página da Medicina Canábica
404.html            Página de erro personalizada
css/                Estilos (styles.css + canabidiol.css)
js/                 main.js (menu, animações, lightbox, FAQ)
assets/img/         Fotos, logo e favicon
assets/video/       hero.mp4 + poster
vercel.json         Cabeçalhos de cache e segurança
sitemap.xml         Mapa do site / robots.txt
```

## Como atualizar o site
1. Edite os arquivos localmente.
2. `git add .` → `git commit -m "descrição"` → `git push`
3. A Vercel publica automaticamente em alguns segundos.
